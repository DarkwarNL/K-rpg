Natuurmuseum Fryslan Potvis Kinect Game

This assignment was for Natuurmuseum Fryslan. They wanted a game that would help children understand and learn where the bones of a sperm whale are. With the help of the artists at Kweekvijvernoord we've created a kinect game in Unity, we chose a kinect game because children would rather play a game with interesting tools then just a standart game. For this project I was the only programmer.

The purpose of the game is that you put the bones on the right place, once you've done this correctly the sperm whale will start swimming away as an reward for finishing the game correctly.

  • Date: September 2016
  • Client: Natuurmuseum Fryslan
  • Category: Game Developer
